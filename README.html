Contextual

Overview

Contextual is a library enabling the implementation of abstractions like functors and monads in a generic way that decouples resolution of operators from operand types. This method is particularly well suited to dynamically typed languages, and languages that do not employ equational reasoning for method resolution, but it also provides significant increase of flexability.

Abstractions

Functors

FMAP

FMAP is used to apply a function of plain values to an embelished value.

Example:

(ctx-run +list+ 
  (fmap #'1+ '(1 2 3)))

LET-FUN

LET-FUN provides parallel functor binding syntax, similar to LET except the values are bound

Example:

(ctr-run +list+
 (let-fun ((x '(1 2))
           (y '(3 4)))
   (+ x y)))
;; => '((4 5) (5 6))

~LET*-FUN~

~LET*-FUN~ provides sequential functor bindind, where the previously bound variable are available in the next expression

Example:

(ctx-run +optional+
  (let*-fun ((x (some 3))
             (y (some 4)))
    (+ x y)))
;; => (some (some 7))(ctx-run +optional+
